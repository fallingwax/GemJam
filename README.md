# GemJam

A Columns clone project for JavaFX. 

##Gameplay Controls: 
A - Move Left  
D - Move Right  
S - Move Down  
W - Drop Piece  
SPACE - Sort Gems  
CTRL - Pause  
Esc - Main Menu  

##Enter High Score Inititals:  
UP - Change Letter Backwards  
DOWN - Change Letter Forwards  
RIGHT - Next Letter  
ENTER - Finish
